Task Manager Web application :
             A simple and user - friendly Task Manager. Using this application users can add, manage, complete with deadline remainders and delete daily tasks.
             
Features :
        - Add new task
        - Mark tasks as completed
        - Delete tasks
        - Filter tasks where users can see all tasks , completed tasks and pending tasks
        - At top of the application, users can also see how many they have and how many tasks they completed and how many tasks thay pending.
        - Clean and responsive Design
        - DOM manipulation Using Javascript
        
Technologies :
         HTML,
         CSS,
         JavaScript.
